# InkTime_Project_TSC
# Descriere Proiect
- Dispozitiv de tip wearable bazat pe microcontroller-ul nRF52840, echipat cu un ecran E-Paper (27x27mm Active Area) și feedback haptic prin motor de vibrație. Alimentarea și programarea se realizează prin mufă USB Type-C 16-pin, având protecție cu diodă MBR0530. PCB-ul este proiectat pe 4 straturi pentru integritate RF (Bluetooth), utilizând componente miniaturizate 0402/0201 și tehnica via stitching pentru ecranare GND.
